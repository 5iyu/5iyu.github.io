部署到github

  1. 新建一个仓库，名字为：github用户名.github.io
  2. 配置git公钥上传到github的setting -> ssh中
  3. hexo g -d 编译静态网页并上传到github仓库
  4. 设置github仓库的setting -> github pages -> source 选择 master branch
  5. 访问github用户名.github.io
